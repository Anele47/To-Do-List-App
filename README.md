# To-Do List Web App

## Overview
The To-Do List Web App is a task management application designed to help users organize and track their daily activities. The application provides an easy-to-use interface for creating, managing, and monitoring tasks. It is built using HTML, CSS, and JavaScript and focuses on improving productivity through task organization and progress tracking.

## Features

### User Authentication
- Login page for user access
- Forgot Password functionality

### Dashboard
- Welcome message for users
- Overview of today's tasks
- Task progress tracking
- Completed tasks summary
- Quick navigation to other pages

### Task Management
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Search tasks
- Filter tasks by:
  - All Tasks
  - Completed Tasks
  - Pending Tasks
  - Important Tasks

### Settings
- Theme customization
- Notification preferences
- User account settings
- Application preferences

### Additional Features
- Responsive design for desktop and mobile devices
- Navigation between all pages
- Task completion percentage tracking
- Productivity tips and motivational quotes
